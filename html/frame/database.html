4.데이터베이스의 종류


















































1.데이터베이스 시스템

        
    
          
  • 데이터베이스 시스템은 데이터를 관리하고 사용자가 데이터에 접근할 수 있도록 도와주는 소프트웨어이다.
    • 
          
  • 이 소프트웨어는 데이터를 저장하고 필요한 때에 찾아주며, 여러 사용자가 동시에 사용할 수 있도록 해준다.
    • 
          
  • 이러한 시스템은 기업의 업무 처리, 은행 거래, 인터넷 서비스, 소셜 미디어 플랫폼 등 다양한 응용 분야에서 사용된다.
    •

2.데이터베이스 관리 시스템

        
    
          
  • 테이블은 데이터를 구조화하고 관리하기 위한 기본 단위이며, 각 열은 속성을 나타낸다.
    • 
          
  • 데이터베이스 관리 시스템(DBMS)은 데이터베이스를 관리하고 다루는 소프트웨어이다.
    • 
          
  • 이 소프트웨어를 사용하면 데이터를 쉽게 추가, 수정, 삭제할 수 있다.
    • 
          
  • 데이터를 안전하게 보호하고 여러 사용자가 함께 작업할 수 있도록 한다.
    • 
          
  • 대표적인 관계형 데이터베이스로는 Oracle, MySQL, SQL Server 등이 있다.
    •

3.SQL(Structured Query Language)

        
    
          
  • SQL은 데이터베이스에서 데이터를 관리하기 위한 표준화된 언어이다.
    • 
          
  • 데이터베이스에 대한 쿼리(Query), 수정, 삭제, 삽입 등의 작업을 수행할 수 있다.
    •

4.데이터베이스의 종류

    
          
  • 관계형 데이터베이스(Relational Database): 테이블 형태로 데이터를 저장하고 관리하는 데이터베이스이다.
    • 
          
  • 이 데이터베이스는 테이블 간의 관계를 정의하고, SQL이라는 언어를 사용하여 데이터를 다룬다.
    • 
          
  • 예를 들어, 학생 데이터베이스 시스템에서 학생 테이블과 성적 테이블이 있을 수 있다.
    • 
        

        
    
          
  • NoSQL 데이터베이스(Not Only SQL): 관계형 데이터베이스와 다르게 유연한 구조를 가진 데이터베이스이다.
    • 
          
  • 이 데이터베이스는 대량의 데이터를 다루거나 빠른 속도로 데이터를 처리할 때 유용하다.
    • 
          
  • 예를 들어, 소셜 미디어에서 사용자의 피드를 저장하고 관리하는 데 NoSQL 데이터베이스를 사용할 수 있다.
    • 
          
  • 이러한 데이터베이스 시스템과 DBMS는 우리 주변에서 많이 사용되며, 많은 응용 프로그램과 시스템에 필수적이다.
    •
